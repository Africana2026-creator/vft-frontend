# Victoria Falls Transporters Website

A professional and creatively unique website for Victoria Falls Transporters, a tourism company specializing in transportation services around Victoria Falls.

## Features

- Responsive design that works on all devices
- Smooth scrolling navigation
- CSS transitions and animations
- Professional color scheme inspired by nature and adventure
- Sections for Home, About, Services, Tours, and Contact

## Structure

- `index.html` - Main HTML file
- `css/style.css` - Stylesheet with transitions and responsive design
- `js/script.js` - JavaScript for interactivity and animations
- `images/` - Folder for image placeholders (currently text files with image titles)

## How to Run

1. Open `index.html` in a web browser
2. The website is fully static and requires no server

## Customization

- Replace the `.txt` files in the `images/` folder with actual `.jpg` images of the same names
- Update contact information in `index.html`
- Modify colors and styles in `css/style.css`
- Add more functionality in `js/script.js`

## Technologies Used

- HTML5
- CSS3 (with transitions and animations)
- JavaScript (ES6+)

## Browser Support

- Modern browsers (Chrome, Firefox, Safari, Edge)
- Responsive design for mobile and desktop
